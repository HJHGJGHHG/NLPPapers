# Sentence Embedding
&emsp;&emsp;**句向量**是**文本表示**中一个重要领域（私认为是文本表示的最终目的，词向量很多时候最后还是要生成句向量），文本表示广泛应用于召回、聚类、分类、匹配等等很多领域。评价句向量多用句间关系任务完成，如句间相似度、句间语义关系推理、问答对等等。
&emsp;&emsp;我们将句向量模型分为 Before BERT 与基于预训练模型两类。
